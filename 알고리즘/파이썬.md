## 나누기 연산
 ``` python

print(a/b) # 나누기
print(a%b) # 나머지
print(a//b) # 몫
print(a**b) # 거듭제곱

```

## List
C의 Array기능, C++의 vector와 유사함
내부적으로 연결 리스트 자료구조를 채택하고 있어서 append(), remove() 등의 메서드를 지원한다.
```python
a = [1,2,3]
a = list()
a = []

n = 10
a = [0] * 10

a = [1,2,3,4,5,7,8,9]
# 뒤에서 첫 번째 원소
print(a[-1])
# 뒤에서 세 번째 원소
print(a[-3])

```
