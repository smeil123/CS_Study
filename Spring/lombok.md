

# Lombok

자바 개발자들의 필수 라이브러리

> 자바 개발 시 자주 사용하는 코드 Getter, Setter, 기본 생성자, toString 등 어노테이션을 자동 생성해준다.


1. 설치한 뒤 build.gradle에 아래 코드 추가하고 refresh  해서 라이브러리를 내려받는다
```
compile('org.projectlombok:lombok')
```
2. 인텔리제이 > Setting 에서 플러그인으로 설치 및 적용(재시작)
