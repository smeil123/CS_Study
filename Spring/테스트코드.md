## 테스트 코드란?
#### TDD
* 테스트가 주도하는 개발
* 테스트 코드를 먼저 작성하는 것부터 시작
<레드 그린 사이클>
* red : 항상 실패하는 테스트를 먼저 작성하고
* green : 테스트가 통과하는 프로덕션 코드를 작성하고
* refactor : 테스트가 통과하면 프로덕트 코드를 리팩토링

#### 단위 테스트
* TDD의 첫 번째 단계인 기능 단위의 테스트 코드를 작성하는 것
* 리팩토링을 포함하지 않고, 순소하
